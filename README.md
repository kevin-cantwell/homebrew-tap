# Homebrew Tap

Custom Homebrew tap for personal tools by [@kevin-cantwell](https://github.com/kevin-cantwell).

## Usage

```bash
brew install kevin-cantwell/tap/<formula>
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [dotmatrix](https://github.com/kevin-cantwell/dotmatrix) | Convert images and videos to Unicode braille art in the terminal |
